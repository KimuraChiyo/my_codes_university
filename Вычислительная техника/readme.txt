Краткая инструкция для загрузки на компьютер среды EWB

Прикладной пакет EWB представлен в zip-архиве в двух вариантах:
1) обычная установка (файл EWB 5.12.zip)
2) режим "portable", т.е. без установки (файл EWB512 (portable).zip)

1-й вариант.
zip-архив содержит 3 файла дистрибутива EWB.
После разархивирования необходимо запустить файл setup.exe и далее следовать инструкциям
во всплывающих диалоговых окнах, т.е. как обычно при установке прикладных программных пакетов.

2-й вариант.
zip-архив содержит уже развернутую среду EWB, т.е. предварительной установки не требуется.
Более того, программа может запускаться даже с флешки, без необходимости предварительного копирования на диск компьютера
(такой режим работы называется "portable").
Запуск среды EWB производится двойным щелчком по исполняемому файлу WEWB32.exe

Примечание 1.
-------------
Поскольку данная среда EWB является разработкой, которая первоначально предполагала работу в операционной среде Windows XP,
перед первым запуском необходима предварительная настройка для возможности  работы в операционных системах Windows 7/8/10/11.

Для этого необходимо щелкнуть правой кнопкой мыши по файлу или ярлыку WEWB32.exe.
В появившемся контекстном меню выбрать строку "Свойства", вкладку "Совместимость".
Далее выбрать опцию (т.е. проставить галочку) в строке "Запустить программу в рекжиме совместимости с:" и выбрать 
режим "Windows XP (Пакет обновления 2)". Обычно этот режим уже высвечивается по умолчанию.
Далее подтвердить выбор нажатием на кнопку "Применить".

Примечание 2.
-------------
Ввиду не полной совместимости графических растров в операционных системах Windows XP и Windows 7/8/10/11,
в некоторых случаях происходит неполная прорисовка изображений передних панелей виртуальных измерительных приборов.
В этом случае надо просто обновить отображение окна на экране монитора,
т.е. выполнить последовательно действия "Свернуть окно"/"Развернуть окно".
